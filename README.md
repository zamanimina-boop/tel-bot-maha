# 🐟 ماهی — ربات هوشمند تلگرام

## ساختار فایل‌ها
```
mahi_bot/
├── bot.py          ← فایل اصلی ربات
├── stock.py        ← تحلیل بورس (TSETMC)
├── summary.py      ← خلاصه‌سازی گفتگو
├── memory.py       ← ذخیره پیام‌ها
├── requirements.txt
└── railway.toml    ← تنظیمات Railway
```

## راه‌اندازی

### ۱. توکن‌های لازم
- **TELEGRAM_TOKEN** → از @BotFather بگیر
- **ANTHROPIC_API_KEY** → از console.anthropic.com بگیر

### ۲. تنظیم BotFather
```
/setprivacy → Disable   (برای خوندن پیام‌های گروه)
/setjoingroups → Enable
```

### ۳. دیپلوی روی Railway
1. یه repo جدید در GitHub بساز
2. فایل‌ها رو push کن
3. در railway.app پروژه جدید از GitHub بساز
4. در Variables این دو رو اضافه کن:
   - `TELEGRAM_TOKEN = توکن_ربات`
   - `ANTHROPIC_API_KEY = کلید_انتروپیک`
5. Deploy کن ✅

### ۴. اضافه کردن به گروه
1. ربات رو سرچ کن
2. به گروه اضافه کن
3. ادمینش کن (برای حذف پیام و غیره)

## دستورات ربات
| دستور | کار |
|-------|-----|
| `/start` | معرفی ربات |
| `/stock خودرو` | تحلیل سهم |
| `/summary` | خلاصه امروز |
| `/help` | راهنما |

## صدا زدن ربات در گروه
- `ماهی، سوال دارم`
- `@نام_ربات سلام`
- Reply کردن به پیام ربات
